# 守望国服大盘 — 设计构思

## 三种设计方向

### 方向 A：Terminal Finance（终端金融）
深色终端风格，绿色数据流，像彭博终端遇上守望先锋。
**概率：0.07**

### 方向 B：OW HUD Reborn（守望 HUD 重生）
直接复刻守望先锋游戏内 HUD 的视觉语言：深蓝底色、青蓝色六边形纹理、橙色高亮、粗体无衬线字体、斜切角边框。
**概率：0.09**

### 方向 C：Stock Exchange × Overwatch（股市 × 守望）
将金融大盘的信息密度与守望先锋的科幻美学融合：深空蓝背景、橙金色高亮、六边形装饰元素、数据面板感。
**概率：0.04**

---

## 选定方向：B — OW HUD Reborn

### Design Movement
守望先锋 HUD 风格 × 金融数据可视化

### Core Principles
1. **深蓝底色 + 青蓝发光**：背景使用深海军蓝（#0A0E1A），主要交互元素使用守望先锋标志性的青蓝色（#00B4D8）发光效果
2. **斜切角边框（Clip-path）**：卡片和按钮使用斜切角设计，复刻 OW 界面的科幻感
3. **六边形装饰纹理**：背景和分隔线使用半透明六边形网格
4. **橙色强调色**：胜率上升/正面指标使用守望先锋橙色（#F97316），连败/负面指标使用红色

### Color Philosophy
- **背景**：#0A0E1A（深空蓝）
- **卡片**：#0D1526（略亮的深蓝）
- **主色**：#00B4D8（守望先锋青蓝）
- **强调橙**：#F97316（OW 橙）
- **胜利绿**：#22C55E
- **失败红**：#EF4444
- **文字主色**：#E2E8F0
- **文字次色**：#94A3B8

### Layout Paradigm
左侧固定导航栏（游戏风格侧边栏）+ 右侧主内容区，非居中布局。顶部有状态栏显示当前赛季和样本量。

### Signature Elements
1. **斜切角卡片**：所有卡片使用 `clip-path: polygon(0 0, calc(100% - 12px) 0, 100% 12px, 100% 100%, 12px 100%, 0 calc(100% - 12px))` 
2. **发光边框**：使用 `box-shadow: 0 0 15px rgba(0, 180, 216, 0.3)` 的青蓝色辉光
3. **六边形背景纹理**：SVG 六边形网格作为背景装饰

### Interaction Philosophy
- 悬停时卡片边框发光增强
- 数字变化时有计数动画
- 图表数据加载时有从左到右的扫描线动画

### Animation
- 页面进入：元素从下方 20px 淡入，错开 80ms
- 数字：从旧值滚动到新值（200ms ease-out）
- 卡片悬停：边框辉光从 0.3 到 0.6 opacity（150ms）
- K 线图：从左到右绘制动画

### Typography System
- **标题字体**：Rajdhani（科幻感粗体）
- **数字字体**：Orbitron（游戏数字感）
- **正文字体**：Noto Sans SC（中文正文）
- 标题：Rajdhani Bold 24-32px
- 指数数字：Orbitron Bold 36-48px
- 正文：Noto Sans SC 14px

### Brand Essence
守望国服大盘 — 为守望先锋玩家提供排位情绪指数的社区数据平台，让每次上分决策有据可依。
**个性**：精准、冷静、有力

### Brand Voice
- 标题示例："先看大盘，再上分"
- CTA 示例："查看今日天气"
- 禁止："欢迎来到守望大盘"

### Wordmark & Logo
六边形内嵌上升箭头 + K 线图形，青蓝色，无文字

### Signature Brand Color
#00B4D8（守望青蓝）
